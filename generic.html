Chatty

July 2023

"Chatty" is a social media app developed using Xcode and SwiftUI, offering an intuitive and streamlined user experience. This app is meant to be a one-stop-shop allowing users to access posts from most major social media platforms in one app. The app features advanced search capabilities for quick content discovery and interactive user engagement tools. Key elements include the Reaction to post feature and comments section, both of which utilize Firebase for persistent data storage. Taking online safety into consideration, an encryption feature is included so only permitted users may log into the user account.

Piggy

August 2024

"Piggy" is a mobile app built with React Native, designed to help users track expenses and manage savings goals effectively. It features a user-friendly interface for monitoring spending and setting financial targets. The app includes advanced security measures such as user authentication and data encryption to ensure the safety of financial information. Key features also include an AI chatbot powered by Vertex AI for personalized financial insights, and a forecasting feature that utilizes past data to predict future expenditures.
